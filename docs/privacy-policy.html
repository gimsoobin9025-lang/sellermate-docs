SellerMate Privacy Policy

Last updated: 2026-03-04

1. 수집하는 데이터

  • 사용자가 도구 실행 시 입력한 상품/키워드 관련 텍스트
  • 서비스 운영 로그(요청 시각, 에러 유형, 런타임 상태 등)
  • 기술적 메타데이터(세션 식별자, 응답 상태 코드)

2. 사용 목적

  • MCP 도구 결과 생성 및 품질 개선
  • 오류 분석, 성능 모니터링, 보안 점검
  • 서비스 안정성 및 사용자 지원

3. 보관 기간

  • 운영 로그: 최소 보관 원칙에 따라 필요한 기간 동안만 저장
  • 법적 의무가 있는 경우 관련 법령에 따라 별도 보관

4. 제3자 공유

원칙적으로 개인정보를 제3자에게 판매/임대하지 않습니다. 단, 서비스 제공을 위한 인프라/AI API 제공자에 최소 범위로 처리될 수 있습니다.

5. 삭제 요청 방법
